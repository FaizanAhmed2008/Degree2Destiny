# Hero Video Setup

## Video Requirements

To add a hero video to the landing page:

1. **Video Format**: MP4 format is recommended for best browser compatibility
2. **Video Location**: Place your video file in this directory (`public/videos/`)
3. **Video Name**: Name it `hero-video.mp4`
4. **Video Specifications**:
   - Recommended resolution: 1920x1080 (Full HD)
   - Recommended duration: 10-30 seconds (looping)
   - File size: Keep under 5MB for optimal loading
   - Codec: H.264 for best compatibility

## Video Content Suggestions

The hero video should show:
- Professional employees working
- Modern office environment
- People collaborating
- Technology/workplace scenes
- Positive, energetic atmosphere

## Free Video Resources

You can find free stock videos at:
- Pexels Videos: https://www.pexels.com/videos/
- Pixabay Videos: https://pixabay.com/videos/
- Unsplash Videos: https://unsplash.com/videos

Search for keywords like:
- "professional work"
- "office collaboration"
- "team working"
- "business meeting"
- "modern workplace"

## Fallback

If no video is provided, the landing page will display a beautiful background image as a fallback.
